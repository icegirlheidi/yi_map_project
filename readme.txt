The webpage is tested on Firefox 36.0.1 with a 13-inch MacBook Pro

Main functionalities:

1. When firstly loading the page, your current location is showned on google map

2. When selecting the campus and place, the place is shown on google map.

3. When selecting the campus and palce, the detailed address and floor is shown on by text on the right side of page.

4. When dragging the google map to the left, it will change back to the centered place.


Technology used: HTML5, CSS3, JavaScript, Google map API
